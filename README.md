## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Framework | ASP.NET Core MVC — .NET 10 |
| Language | C# |
| Views | Razor (.cshtml) |
| Styling | Custom CSS3 per-page (no Bootstrap UI) |
| Fonts | Google Fonts — DM Sans, Cormorant Garamond, Montserrat |
| Architecture | MVC Pattern |
| Validation | Data Annotations + jQuery Unobtrusive Validation |

```

├── Controllers/
│   ├── Admin.cs                    # Dashboard, Bookings, Vehicles, Customers, Profile
│   ├── AuthenticationController.cs # Login, Register
│   └── HomeController.cs           # Index, Car_view, Car_rent, Profile, Contact
├── Models/
│   ├── CarRentViewModel.cs
│   ├── VehicleViewModel.cs
│   ├── LoginViewModel.cs
│   ├── RegisterViewModel.cs
│   ├── ContactViewModel.cs
│   ├── UserProfileViewModel.cs
│   ├── AdminProfileViewModel.cs
│   └── ...
├── Views/
│   ├── Admin/          # Dashboard · Adminbookings · AdminVehicles · AdminCustomers · AdminProfile
│   ├── Authentication/ # Login · Register
│   ├── Home/           # Index · Car_view · Car_rent · Profile
│   └── Shared/         # _Layout · Error
└── wwwroot/
    └── css/            # Per-page stylesheets (home · carview · carrent · dashboard · ...)
```

## 🚀 Getting Started

### Prerequisites
- [.NET 10 SDK](https://dotnet.microsoft.com/download)

### Run Locally

```bash
git clone https://github.com/<your-username>/LUXURY_DRIVE.git
cd LUXURY_DRIVE
dotnet run
```

> App starts at `https://localhost:5001` — default route opens the **Login** page.

| URL | Page |
|---|---|

| `/Home/Car_rent` | 📅 Book a Vehicle |
